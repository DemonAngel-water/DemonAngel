8.右键添加快捷方式
	近来一直用Typora这个神器写东西，感觉相当好，但新建一个文件的时候总是要右键新建一个txt文件，然后再改后缀，这样子还蛮烦的，因此最好的做法是右键直接就能新建一个md文件。实现如下，新建一个txt文件，填入如下内容：

	Windows Registry Editor Version 5.00
	[HKEY_CLASSES_ROOT\.md]
	@="Typora.exe"
	[HKEY_CLASSES_ROOT\.md\ShellNew]
	"NullFile"=""
	[HKEY_CLASSES_ROOT\Typora.exe]
	@="Markdown"

	
	然后把文件后缀改成reg，双击一下就行了。
	当然这个方法也可以延伸一下，只要把“template.md”改成别的，那么就能建别的格式的文件

11、列操作
	word中
		按下alt键，竖向选择文本内容，选中后按delete删除即可
	sublime text中
		1.选择需要列操作的文本
		2.按下ctrl + shift + l
		3.操作即可
	notepad++中（不能单独在每行行尾添加内容）
		alt + 鼠标左键选中区域
	notepad++（单独在每行行尾添加内容）
		Ctrl + H 唤起 替换工具栏
		
		查找目标: $
		替换为: 需要添加的文字（如果存在转义字符使用/）
		注: ^ 为行尾	$ 为行尾
		
		查找模式: 正则表达式

16、Windows10系统安装Microsoft Store
	关于 “安装 Microsoft Store” 的问题，正常情况下 Windows 是会内置 Microsoft Store 的，如果您在系统中没有找到微软商城应用，可能是由于您安装的是精简版的 Windows。
	按 “Windows 徽标键+X”，启动 “Windows PowerShell（管理员）”，执行以下两条命令：
	get-appxpackage *store* | remove-Appxpackage
	add-appxpackage -register "C:\Program Files\WindowsApps\*Store*\AppxManifest.xml" -disabledevelopmentmode
	执行完毕后重启设备，看一下能否成功安装 Microsoft Store。
